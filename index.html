HTML Tables Demystified

HTML Table Tags
Element Attribute Definition Syntax/Link/Attribute Values
<table> Defines a table W3 Schools link
<thead> Groups the header content in a table
<tbody> Used in conjunction with thead and tfooter, it groups the body content allowing independent scrolling W3 School example
<tfoot> Used to group footer content in conjunction with the thead and tbody elements to specify each part of a table. W3 School example
<th> Defines a header cell in a table abbr, colspan, headers, rowspan, scope
<tr> Defines a row in a table W3 School example
<td> Defines a cell in a table colspan, headers, rowspan
<caption> Defines a table caption W3 School example
<colgroup> Specifies a group of one or more columns in a table for formatting. W3 School example
<col> Specifies column properties for each column within a colgroup span
abbr Specifies an abbreviated version of the content in a header cell which is utilised by some screen readers <th abbr="Make">Toy manufacturer</th>
colspan This attribute contains a non-negative integer value that indicates for how many columns the cell extends. Its default value is 1. <th colspan="2">Monthly Savings</th>
headers Specifies the number of columns a header cell should span <th colspan="2">Monthly Savings</th>
rowspan Specifies whether a header cell is a header for a column, row, or group of columns or rows <th rowspan="3">Savings for Holiday</th>
scope The scope attribute specifies whether a header cell is a header for a column, row, or group of columns or rows. col, row, colgroup, rowgroup

HTML Forms
Element Tag Attribute Tag Description Syntax/Useful Link/Attribute values
<form> Defines an HTML form for user input action, target, method
action The action attribute defines the action to be performed when the form is submitted. Usually, the form data is sent to a page on the server when the user clicks on the submit button. <form action="/action_page.php">
target The target attribute specifies if the submitted result will open in a new browser tab, a frame, or in the current window. <form action="/action_page.php" target="_blank">
method The method attribute specifies the HTTP method (GET or POST) to be used when submitting the form data. <form action="/action_page.php" method="get">
<input> The input element is the most important form element. The input element is displayed in several ways, depending on the type attribute. MDN weblink to huge list of types and variables
<label> To associate the label with an input element, you need to give the input an id attribute. The MDN webdoc for HTML labels
<select> The select element is used to create a drop-down list. The option tags inside the select element define the available options in the list. W3 Schools example link
<datalist> The datalist tag is used to provide an "autocomplete" feature on input elements. Users will see a drop-down list of pre-defined options as they input data. Use the input element's list attribute to bind it together with a datalist element. W3 Schools example link
<textarea> The textarea> tag defines a multi-line text input control. The size of a text area can be specified by the cols and rows attributes, or even better; through CSS' height and width properties. W3 Schools example link
